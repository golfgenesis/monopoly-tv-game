# 📺 ติดตั้งและเล่นบน TCL 75″ 4K Mini QLED Google TV (รุ่น 75Q6C)

เอกสารนี้อธิบายวิธีเอา **เศรษฐีสยาม** ขึ้นจอ TCL 75Q6C โดยเฉพาะ — ทีวีจอ HVA 4K (3840×2160) ระบบ Google TV (Android TV) Mini QLED

ทีวีทำหน้าที่เป็น **กระดานกลาง** เท่านั้น ไม่ต้องลงแอป ไม่ต้องพิมพ์ระหว่างเล่น แค่เปิดหน้าเว็บ host ค้างไว้ ที่เหลือเล่นบนมือถือ

---

## ภาพรวมสถาปัตยกรรม

```
   [ คอม/โน้ตบุ๊ก ]  ──รัน──>  server + TV + phone (npm run dev)
          │  Wi-Fi / LAN วงเดียวกัน
          ├────────────────► 📺 TCL 75Q6C  เปิด http://<ip-คอม>:5173  (กระดาน)
          ├────────────────► 📱 มือถือคนที่ 1  เปิด http://<ip-คอม>:5174
          └────────────────► 📱 มือถือคนที่ 2..6
```

> **สำคัญ:** ต้องมี "เครื่อง host" สักเครื่อง (คอม/โน้ตบุ๊ก/มินิพีซี/Raspberry Pi) ที่รัน `npm run dev` อยู่ในวง Wi-Fi เดียวกับทีวีและมือถือ ตัว Google TV เองไม่ได้รันเซิร์ฟเวอร์ — มันแค่เปิดหน้าเว็บ

---

## ขั้นที่ 1 — เตรียมเครื่อง host

บนคอม (Windows/Mac/Linux):

```bash
git clone <repo> && cd monopoly-tv-game
npm install
npm run dev
```

มองหาบรรทัด `Network:` ที่ขึ้นมา เช่น

```
[tv]    ➜  Network: http://192.168.1.50:5173/
[phone] ➜  Network: http://192.168.1.50:5174/
```

จด IP นี้ไว้ (ในตัวอย่างคือ `192.168.1.50`) — จะใช้ทั้งบนทีวีและมือถือ

> ถ้า Windows เด้งถาม Firewall ให้กด **Allow access** ให้ Node.js (ทั้ง Private/Public network) ไม่งั้นทีวี/มือถือจะต่อไม่ได้

## ขั้นที่ 2 — เปิดหน้า host บน TCL Google TV

Google TV **ไม่มีเบราว์เซอร์ติดตั้งมาให้** ตั้งแต่แรก เลือกวิธีใดวิธีหนึ่งตามสะดวก (เรียงจากแนะนำสุด):

### วิธี A — ต่อ HDMI จากเครื่อง host (เสถียร + ภาพคมสุด) ⭐ แนะนำ

1. ต่อสาย HDMI จากคอม/มินิพีซีเข้าทีวี เลือก input HDMI นั้น
2. เปิด Chrome/Edge บนคอม ไปที่ `http://localhost:5173`
3. กด **F11** เพื่อเข้าฟูลสกรีน
4. ทีวีกลายเป็นจอแสดงผล กระดานเต็มจอ 4K ทันที

ข้อดี: ประสิทธิภาพดีสุด, เสียงทำงานเลย, ไม่ต้องพึ่งเบราว์เซอร์บนทีวี

### วิธี B — Cast แท็บ Chrome ขึ้นทีวี (ไร้สาย ง่าย)

TCL 75Q6C มี **Chromecast built-in**

1. บนคอมเปิด Chrome → `http://192.168.1.50:5173`
2. เมนู Chrome (⋮) → **Cast…** → เลือกทีวี TCL → **Cast tab**
3. กดไอคอนฟูลสกรีนในแท็บ

ข้อดี: ไม่ต้องเดินสาย / ข้อเสีย: ภาพ/เสียงผ่านการบีบอัด อาจมีดีเลย์เล็กน้อย

### วิธี C — เบราว์เซอร์บนทีวีโดยตรง (สแตนด์อโลนสุด)

ลงเบราว์เซอร์บน Google TV แล้วเปิด URL ตรงๆ:

1. เปิด **Google Play Store** บนทีวี ค้น "browser" — ตัวที่ใช้ได้ดีเช่น **TV Bro**, **Puffin TV** (ความพร้อมแล้วแต่ภูมิภาค)
   - ถ้าหาในสโตร์ไม่เจอ สามารถ *sideload* APK ของ **TV Bro** (โอเพนซอร์ส) ได้
2. เปิดเบราว์เซอร์ → พิมพ์ `192.168.1.50:5173` ในช่อง URL
3. ใช้รีโมตเลื่อน แล้วกด **OK** หนึ่งครั้ง (เพื่อปลดล็อกเสียง — ดูหมายเหตุด้านล่าง)

ข้อดี: ทีวีทำงานเองล้วน ไม่ต้องมีจอ HDMI / ข้อเสีย: เบราว์เซอร์บนทีวีแต่ละตัวคุณภาพ/ความเร็วต่างกัน

## ขั้นที่ 3 — เชื่อมมือถือ

1. ทีวีจะโชว์ **QR code** + **รหัสห้อง 5 ตัว** เอง
2. ผู้เล่นสแกน QR ด้วยกล้องมือถือ → เปิดหน้า controller พร้อมรหัสห้องอัตโนมัติ
   - หรือเปิด `http://192.168.1.50:5174` แล้วพิมพ์รหัสห้องเอง
3. เลือกตัวละคร + สี + ชื่อเล่น → **เข้าร่วมเกม**
4. มี 2–6 คนแล้ว กด **เริ่มเกม** บนทีวี

## 🤖 เล่นคนเดียวกับบอท

ไม่มีเพื่อน? เล่นกับ **บอท AI** ได้เลย ทุกอย่างคุมจากหน้าจอทีวีในล็อบบี้

### ขั้นตอน

1. เปิด TV host ตามปกติ (ทีวีสร้างห้อง + โชว์ QR)
2. ที่แผงควบคุมล่างขวาของทีวี (ตอนอยู่ในล็อบบี้) จะมีปุ่ม:
   - **🤖 เพิ่มบอท** — กดเพื่อเพิ่มบอท 1 ตัว (กดซ้ำได้ รวมผู้เล่น+บอท สูงสุด 6)
   - **ลบบอท (n)** — เอาบอทตัวล่าสุดออก
3. เข้าร่วมด้วยมือถือของคุณเอง 1 เครื่อง (สแกน QR)
4. มีรวม 2 คนขึ้นไป (เช่น คุณ 1 + บอท 1) → กด **เริ่มเกม**

> **เล่นคนเดียวล้วน:** เพิ่มบอท 1–5 ตัวแล้วเริ่มได้เลย / บอทกับเพื่อนปนกันก็ได้

### กดปุ่มบนทีวียังไง (ไม่มีเมาส์)

- **รีโมต Google TV:** ใช้ปุ่มลูกศร (D-pad) เลื่อนโฟกัส — ปุ่มที่เลือกอยู่จะมี **กรอบสีทอง** — แล้วกด **OK** เพื่อกด
- **ต่อ HDMI + เมาส์/ทัชแพด:** คลิกได้ตรงๆ
- **Cast จากคอม:** กดบนแท็บ Chrome ที่คอมได้เลย

### บอทเล่นยังไง

บอทตัดสินใจที่เซิร์ฟเวอร์ เว้นจังหวะ "คิด" ~1.2 วิ/การกระทำ เพื่อให้ดูทันบนทีวี:
ทอยเต๋า · ซื้อที่ดิน (เก็บเงินสดสำรองไว้บ้าง, ยอมจ่ายแพงขึ้นถ้าจะได้ครบโซน) · ลงประมูล · สร้างบ้านเมื่อถือครบโซนและเงินเหลือ · จัดการเรื่องคุก · **ตอบรับข้อเสนอเทรดที่คุ้มค่า** (คุณยื่นเทรดให้บอทได้!)

## 📺📺 เล่นบนทีวีเครื่องอื่น เช่น Sony KD-55X8500D ได้ไหม?

**ได้** — จอกระดานเป็นแค่หน้าเว็บ เปิดบนเบราว์เซอร์อะไรก็ได้ที่อยู่ Wi-Fi วงเดียวกัน แต่ต้องเข้าใจ 2 เรื่อง:

### 1) หนึ่งเกม = หนึ่งจอกระดาน

ทีวีแต่ละเครื่องที่เปิด TV host จะ **สร้างห้องของตัวเอง** (คนละรหัส) ดังนั้น:
- ใช้ Sony **แทน** TCL เป็นจอกระดานได้สบาย
- แต่ **ไม่สามารถ** ให้ทั้ง TCL และ Sony โชว์ "เกมเดียวกัน" พร้อมกันได้ (ยังไม่มีโหมดมิเรอร์/ดูร่วม) — ถ้าเปิด 2 เครื่องจะกลายเป็น 2 เกมแยกกัน

### 2) จะกระตุกไหม? — ขึ้นกับ "วิธีเปิด"

KD-55X8500D เป็นทีวี Android ปี 2016 ชิปรุ่นเก่า เบราว์เซอร์ในตัวค่อนข้างช้า เกมนี้มีเอฟเฟกต์ที่กิน GPU (หมากเลื่อน, ลูกเต๋าหมุน, เงา, เบลอ) ดังนั้น:

| วิธีเปิดบน Sony | ลื่นไหม | คำแนะนำ |
| --- | --- | --- |
| **HDMI จากคอม/โน้ตบุ๊ก** → Chrome ฟูลสกรีน | ⭐ ลื่นมาก | **แนะนำสุด** — ทีวีเก่าแค่ไหนก็ไม่กระตุก เพราะคอมเรนเดอร์ |
| **Cast แท็บ Chrome** จากคอมขึ้น Sony | 🆗 ดี | มีดีเลย์/บีบอัดนิดหน่อย แต่ลื่นกว่าเบราว์เซอร์ในทีวี |
| **เบราว์เซอร์ในตัว Sony** เปิด URL ตรงๆ | ⚠️ อาจกระตุก | ทำได้ แต่ฮาร์ดแวร์ปี 2016 อาจหน่วง โดยเฉพาะตอนหมากเดิน/เปิดโมดัล |

**ถ้าจำเป็นต้องเปิดในเบราว์เซอร์ของ Sony เอง** ให้ลดภาระลง:

1. ในเกม → เมนูซ้าย → **ตั้งค่า** → **ปิดแอนิเมชัน** (หมากจะวาร์ปแทนเลื่อน, ไม่มีลูกเต๋าหมุน → ลื่นขึ้นชัดเจน)
2. ปิด **เสียง** ถ้าเบราว์เซอร์ในทีวีจัดการเสียงได้ไม่ดี
3. ตั้งทีวีเป็นโหมดภาพ **Game / Graphics** เพื่อลดดีเลย์

> สรุป: อยากชัวร์ว่าลื่นบน Sony รุ่นนี้ → ใช้ **HDMI จากคอม** ดีที่สุด ส่วน TCL 75Q6C (ชิปใหม่กว่ามาก) เปิดในเบราว์เซอร์ของทีวีเองได้สบายกว่า

## ปรับจอให้พอดีทีวี (Overscan / ขอบภาพโดนตัด)

ทีวีบางเครื่อง/บางโหมดจะ "ซูมเกิน" ทำให้ขอบ UI โดนตัด เกมนี้แก้ได้ 2 ทาง:

1. **ในเกม:** เมนูซ้าย → **ตั้งค่า** → **ปรับขอบจอ (Overscan)** ลดเปอร์เซ็นต์ลงทีละนิด (ค่าเริ่มต้น 98%) จนเห็นขอบครบ ค่าจะถูกจำไว้ในเครื่องนั้น
2. **ที่ทีวี:** ตั้งค่าทีวี → ภาพ → ตั้งอัตราส่วน/ขนาดภาพเป็น **"พอดีหน้าจอ / Just Scan / Screen Fit"** (ไม่ใช่ 16:9 zoom) เพื่อให้แสดงแบบ 1:1 พิกเซล

> UI ออกแบบบนผืน 1920×1080 แล้วสเกลขึ้นเต็มจอแบบยูนิฟอร์ม ดังนั้นไม่ว่าทีวี/เบราว์เซอร์จะรายงานความละเอียดเป็น 4K หรือ 1080p เลย์เอาต์จะเหมือนกันเป๊ะ ต่างแค่ตัวคูณสเกล

## หมายเหตุเฉพาะ Google TV / 75Q6C

- **เสียงเอฟเฟกต์:** เบราว์เซอร์บล็อกเสียงอัตโนมัติจนกว่าจะมี "การกดครั้งแรก" ถ้าใช้วิธี C ให้กดปุ่มรีโมต (OK/ลูกศร) บนหน้าเกม **หนึ่งครั้ง** เพื่อปลดล็อก หรือปิดเสียงไปเลยที่เมนูซ้ายถ้าไม่ต้องการ
- **ความละเอียด viewport:** เบราว์เซอร์บนทีวีบางตัวเรนเดอร์ที่ 1080p แล้วให้พาเนล 4K อัปสเกล — ไม่กระทบ เพราะระบบสเกลพอดีจอรองรับทั้งสองแบบ
- **จอ HVA + Mini QLED:** คอนทราสต์สูง สีจัด เหมาะกับโทนกล่องของเล่นยุค 90 ของเกมนี้พอดี ไม่ต้องตั้งค่าพิเศษ
- **โหมดเกม/PC ของทีวี:** ถ้าใช้วิธี A (HDMI) แนะนำเปิด **Game Mode** หรือตั้ง label อินพุตเป็น "PC" เพื่อลดดีเลย์และปิด overscan อัตโนมัติ
- **รีโมต:** ระหว่างเล่นไม่ต้องใช้รีโมตเลย ทุกการตัดสินใจทำบนมือถือ รีโมตใช้แค่ตอนเปิดหน้า/ปลดล็อกเสียง/เปิดเมนูตั้งค่า

## แก้ปัญหาที่พบบ่อย

| อาการ | วิธีแก้ |
| --- | --- |
| ทีวีเปิด URL ไม่ได้ / หน้าว่าง | เช็กว่าทีวีกับคอมอยู่ Wi-Fi วงเดียวกัน และใช้ **IP คอม** ไม่ใช่ `localhost` |
| มือถือขึ้น "ไม่พบห้องนี้" | พิมพ์รหัสห้องผิด หรือเซิร์ฟเวอร์ถูกรีสตาร์ต (ห้องหาย) — สร้างห้องใหม่บนทีวี |
| ต่อไม่ติดทั้งทีวีและมือถือ | Windows Firewall บล็อก Node.js → อนุญาตในแผง Firewall |
| ขอบภาพโดนตัด | ลด Overscan ในตั้งค่าเกม + ตั้งทีวีเป็น Just Scan/Screen Fit |
| ไม่มีเสียง | กดปุ่มรีโมตบนหน้าเกม 1 ครั้ง (ปลดล็อกเสียง) หรือเปิดเสียงในตั้งค่า |
| มือถือหลุด/ดับแล้วเข้าใหม่ไม่ได้ | ปกติจะ **ต่อกลับเข้าเกมเดิมอัตโนมัติ** ถ้ายังไม่กลับ ให้เปิดหน้า phone ใหม่ด้วยรหัสห้องเดิม |
| IP เปลี่ยนหลังรีสตาร์ตเราเตอร์ | รัน `npm run dev` ใหม่ แล้วใช้ IP จากบรรทัด `Network:` อันใหม่ |
| ภาพกระตุกบนเบราว์เซอร์ทีวี | ปิด **แอนิเมชัน** ในตั้งค่าเกม หรือเปลี่ยนไปใช้วิธี A (HDMI) |

## เคล็ดลับให้เนียนที่สุด

- ใช้ **มินิพีซีตัวเล็ก** (เช่น Intel NUC) หรือโน้ตบุ๊กเก่า ต่อ HDMI ค้างไว้กับทีวี เปิด Chrome คีออสก์โหมด `chrome --kiosk http://localhost:5173` → เปิดทีวีปุ๊บเข้าเกมเลย
- ตั้ง IP ของเครื่อง host เป็น **Static IP / DHCP reservation** ในเราเตอร์ จะได้ไม่ต้องจด IP ใหม่ทุกครั้ง
- พิมพ์รหัสห้องตัวใหญ่ให้แขกเห็นง่าย หรือให้สแกน QR เป็นหลัก
