# 🎲 กติกาเศรษฐีสยาม

เกมซื้อขายที่ดิน 2–6 คน เป้าหมายคือเป็น **เศรษฐีคนสุดท้ายที่ยังไม่ล้มละลาย**

> เล่นคนเดียวได้ด้วยการเพิ่ม **บอท** จากทีวี (ดู [SETUP-TCL-GOOGLE-TV.md](SETUP-TCL-GOOGLE-TV.md#-เล่นคนเดียวกับบอท)) บอทเล่นตามกติกาเดียวกันทุกข้อ

## ลำดับการเล่นหนึ่งตา

1. **ทอยลูกเต๋า 2 ลูก** บนมือถือ → หมากเดินตามผลรวม
2. เกิดผลตามช่องที่หยุด (ซื้อ/จ่ายค่าเช่า/การ์ด/ภาษี/คุก ฯลฯ)
3. ทำรายการเสริมได้ (สร้างบ้าน จำนอง เทรด) — เฉพาะในตาของตัวเอง
4. **จบตา** ส่งต่อคนถัดไป

**แต้มคู่:** ทอยได้เลขเท่ากัน → ได้ทอยซ้ำอีกครั้ง แต่ถ้าทอยแต้มคู่ **3 ครั้งติด** ในตาเดียว → ถูกส่งเข้าคุกทันที

**ผ่านช่องเริ่ม:** ทุกครั้งที่เดินผ่านหรือหยุดที่ช่อง `รับเงินเดือน` รับ **฿2,000**

## ที่ดินและค่าเช่า

หยุดช่องที่ดินที่ยังไม่มีเจ้าของ → เลือก **ซื้อ** ตามราคาหน้าโฉนด หรือ **ข้าม**
ข้าม = เปิด **ประมูล** ให้ผู้เล่นทุกคนแย่งราคากัน (ดูหัวข้อประมูล)

หยุดช่องที่คนอื่นเป็นเจ้าของ → จ่าย **ค่าเช่า** ให้เจ้าของ (ยกเว้นโฉนดที่ถูกจำนองอยู่ = ไม่ต้องจ่าย)

### บันไดค่าเช่า (ที่ดินทั่วไป)

ค่าเช่าคิดจาก `ค่าเช่าฐาน` ของแต่ละช่อง คูณตามนี้:

| สถานะ | ค่าเช่า |
| --- | --- |
| ไม่มีบ้าน, ถือไม่ครบโซน | ×1 (ฐาน) |
| ไม่มีบ้าน, **ถือครบทั้งโซนสี** | ×2 |
| มี 1 บ้าน | ×5 |
| มี 2 บ้าน | ×15 |
| มี 3 บ้าน | ×45 |
| มี 4 บ้าน | ×70 |
| **โรงแรม** | ×95 |

### รถขนส่ง (สถานี/ท่าเรือ)

ค่าเช่าขึ้นกับจำนวนช่องขนส่งที่เจ้าของถือ: 1 ช่อง → **฿250**, 2 → **฿500**, 3 → **฿1,000**, 4 → **฿2,000**

### สาธารณูปโภค (ไฟฟ้า/ประปา)

ค่าเช่า = ผลรวมแต้มเต๋าที่ทอย × ตัวคูณ: ถือ 1 ช่อง → **×40**, ถือทั้ง 2 ช่อง → **×100**

## สร้างบ้านและโรงแรม

- สร้างได้เมื่อ **ถือครบทุกช่องในโซนสีเดียวกัน** และโซนนั้นไม่มีช่องที่ถูกจำนอง
- ต้อง **สร้างเฉลี่ยทั่วโซน** (even build): ห้ามสร้างหลังที่ 2 บนช่องหนึ่ง ก่อนที่ทุกช่องในโซนจะมี 1 หลัง
- บ้านหลังที่ 5 = **โรงแรม**
- ขายคืนได้ครึ่งราคาค่าสร้าง (ต้องขายจากช่องที่มีบ้านมากสุดในโซนก่อน — even sell)

## จำนอง

- ขัดสนเงินสด → **จำนอง** โฉนดเพื่อรับเงินทันที = **ครึ่งหนึ่งของราคาโฉนด**
- โฉนดที่จำนองอยู่จะ **ไม่เก็บค่าเช่า**
- **ไถ่ถอน** ต้องจ่ายคืน = เงินจำนอง **× 1.1** (ดอกเบี้ย 10%)
- จำนองโฉนดในโซนได้เฉพาะเมื่อโซนนั้น **ไม่มีบ้าน** (ต้องรื้อบ้านขายคืนก่อน)

---

## ประมูล

เมื่อมีคน **ข้าม** ไม่ซื้อที่ดินที่เหยียบ → เปิดประมูลทันที:

1. ผู้เล่นทุกคน (เริ่มจากคนที่ข้าม) ผลัดกัน **เสนอราคา** หรือ **ผ่าน**
2. ราคาเสนอต้องสูงกว่าราคาสูงสุดปัจจุบัน (มือถือมีปุ่ม +฿100 / +฿500)
3. เหลือผู้เสนอราคาคนเดียว → คนนั้นได้ที่ดินในราคาที่เสนอ
4. ถ้าทุกคนผ่านหมด → ที่ดินยังว่างต่อไป

## การ์ดดวง 🃏 และ งานบุญ 🎁

หยุดช่อง `ดวง` หรือ `งานบุญ` → เปิดการ์ดสุ่ม 1 ใบ ผลมีตั้งแต่:
รับเงิน / จ่ายเงิน / เลื่อนหมากไปช่องอื่น / รับ-จ่ายกับเพื่อนทุกคน / โดนจับเข้าคุก / ได้บัตรพ้นโทษ

## ติดคุก 🚧

**เข้าคุกได้ 3 ทาง:** หยุดช่อง `ไปคุก!`, ทอยแต้มคู่ 3 ครั้งติด, หรือเปิดการ์ดโดนจับ

**ออกจากคุก:**
- ทอย **แต้มคู่** ในตาของตัวเอง (ออกแล้วเดินตามแต้มเลย แต่ไม่ได้ทอยซ้ำ)
- **จ่าย ฿500**
- ใช้ **บัตรพ้นโทษ** (ได้จากการ์ด)
- ครบ 3 ตายังออกไม่ได้ → ถูกบังคับจ่าย ฿500 แล้วออก

> ช่อง `เยี่ยมคุก` (มุมล่างซ้าย) แค่แวะเฉยๆ ไม่ติดคุก

## ล้มละลายและผู้ชนะ

- จ่ายหนี้ (ค่าเช่า/ภาษี/ค่าปรับ) **ไม่ไหว** → จ่ายเงินสดที่เหลือทั้งหมด แล้ว **ล้มละลาย**
- ทรัพย์สินทั้งหมดโอนให้เจ้าหนี้ (ถ้าหนี้กับธนาคาร = ที่ดินกลับเป็นว่าง)
- เหลือผู้เล่นที่ยังไม่ล้มละลาย **คนสุดท้าย** = 🏆 **เศรษฐีที่ยิ่งใหญ่!**
- ทีวีจะโชว์พิธีประกาศผู้ชนะ + อันดับตาม **มูลค่าสุทธิ** (เงินสด + ราคาที่ดินที่ไม่ได้จำนอง + มูลค่าบ้าน/โรงแรม)

## เกร็ดเชิงเทคนิค (ความยุติธรรม)

- **เซิร์ฟเวอร์ทอยเต๋าและจับการ์ดเอง** — ค่าเต๋า/การ์ดที่มือถือส่งมาถูกเขียนทับด้วยค่าสุ่มจากเซิร์ฟเวอร์เสมอ มือถือจึงโกงไม่ได้
- ทุกแอ็กชันถูกตรวจสิทธิ์ฝั่งเซิร์ฟเวอร์ (เช่น คนที่ไม่ใช่เจ้าของตาทอยเต๋าไม่ได้)
- ข้อมูลส่วนตัว (เงิน/โฉนด/ข้อเสนอเทรด) แสดงบนมือถือเจ้าตัวเท่านั้น
